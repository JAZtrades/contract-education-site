# Contract Education Site 2.0

A redesigned static GitHub Pages website for John Zarcaro focused on private contract education.

## Files

- `index.html` — redesigned Contract Education Site 2.0 landing page
- `styles.css` — professional visual system, responsive layout, and component styling
- `assets/john-photo-hq.webp` — professional portrait used on the home page
- `contract.html` — Scope of Services Agreement page
- `schedule.html` — intro call lead page
- `pay.html` — agreement-first payment process page

## GitHub Pages setup

Use GitHub Pages with the current branch and root folder. If publishing to a new GitHub project, create a new repository named `contract-education-site-2` and push this branch to it.
